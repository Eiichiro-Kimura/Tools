# 習作Tools

## リポジトリの概要

言語の学習も兼ねて、自分が必要としているToolの格納用リポジトリ。
あくまで習作なので、エラー処理は結構適当。

## ツールの概要

### dazn_schedule

Flutterスマホアプリの習作。
DAZNの番組検索、Googleカレンダーへの登録がしたくて作成。
GoogleAPIのクライアントキーが必要（なくても番組検索は可能）。
iPhone12mini / Androidエミュレータで動作確認済み。

### RedmineSupporter

Pythonの習作。
コマンドラインからRedmineの作業時間入力、期日変更、子チケットの取得がしたくて作成。
